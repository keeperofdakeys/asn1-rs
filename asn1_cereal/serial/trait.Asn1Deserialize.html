Provides the methods required to deserialize this Rust type from an ASN.1 stream.

When implementing this for a simple primitive type, implementing deserialize_bytes should be all that's required.

Required Methods

fn deserialize_bytes<E: Asn1EncRules, I: Iterator<Item=Result<u8>>>(e: E, reader: &mut I, len: Option<LenNum>) -> Result<Self, DecodeError>

Deserialise ASN.1 data without a tag into a value.

The data length must be explicitly passed to this function. For primitive types, an error will be returned if this length is Indefinite.
